Image Migration to OSS Tool

This tool will migrate locally stored images to Alibaba Cloud OSS. Please ensure before execution:

  • OSS parameters are correctly configured
  • OSS account has sufficient storage space
  • Recommended to execute during maintenance time to avoid affecting user access
  • Please backup the database before execution
